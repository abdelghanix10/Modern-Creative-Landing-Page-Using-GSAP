# Modern Creative Landing Page Using GSAP

A modern, responsive landing page for Moncler featuring smooth GSAP animations, interactive Swiper carousel, and elegant design.

## Features

- **Smooth Animations**: Powered by GSAP for fluid transitions and effects
- **Interactive Carousel**: Swiper.js for seamless slide navigation
- **Responsive Design**: Optimized for all device sizes
- **Modern UI**: Clean, minimalist design with custom fonts
- **Product Showcase**: Multiple slides showcasing Moncler Genius collection

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with gradients and animations
- **GSAP**: GreenSock Animation Platform for advanced animations
- **Swiper**: Touch-enabled slider library
- **Remixicon**: Icon library for UI elements
- **Google Fonts**: Poppins font family
- **Custom Font**: P22 font for branding

## Setup and Installation

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No additional setup required - all dependencies are loaded via CDN
